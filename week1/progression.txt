ซ้อน object ในนี้มี 2 property คือ
1. name: "mongkol" value เป็นstring ธรรมดา
2. hello: function() { return "Hello" }  value เป็น function
mothod สามารถเก็บเป็น valueได้เพราะใน js functionเป็นfirst class citizen
